# 📝 Todo REST API

Mini project backend sederhana menggunakan **Golang**, **Gin**, **GORM**, dan **PostgreSQL** untuk mengelola data **User** dan **Todo** melalui REST API.

## Features

* User CRUD
* Todo CRUD
* RESTful API
* PostgreSQL Database Integration
* GORM ORM

## Tech Stack

* Go
* Gin
* GORM
* PostgreSQL
